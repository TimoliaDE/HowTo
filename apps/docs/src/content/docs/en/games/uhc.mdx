## Game Concept
Epic battles with up to 100 players and diverse scenarios are guaranteed here! Pick your favorite scenario and fight alone or in a team until the last man standing. Craft your own equipment and fight the battles your way.

<p></p>

## How it works
The goal of the game is to eliminate all opponents and survive as the last individual player or the last team.
Each UHC has its own scenarios (e.g., Fireless = no fire or lava damage; NoClean = 30s protection after a kill; Free-for-all or Teams) that significantly change your environment and the gameplay experience.
These scenarios are announced two hours before the game starts on the [Timolia Discord](https://timolia.de/discord) or the [Twitter account](https://twitter.com/TimoliaUHC).

At the start of the game, all players are spawned on a vanilla map. During the pre-determined protection period, you can prepare for the upcoming battles and farm your equipment in peace.
In the tenth minute of the game, you will be healed one last time.
Once the protection period expires, PvP is activated. From now on, you better watch your back!
As the game progresses, the playable area gets smaller. The border surrounding the field shrinks in stages. If you are outside the border after it shrinks, you will be teleported back onto the field.

## Scenarios
Here you can find all UHC scenarios and how they influence the gameplay.

| Scenario | Description |
| :--- | :--- |
| **1.7 Enchantments** | Enchantments are not visible in the enchantment table. Additionally, you do not need Lapis Lazuli. |
| **Absorptionless** | Golden Heads and golden apples do not grant absorption hearts. |
| **Apple Kill** | Every player receives three golden apples at the end of the protection period, plus two additional golden apples per kill. |
| **Backpack** | Enables a shared inventory in Team UHCs (`/backpack` or `/bp`). |
| **Bandits** | If you hit a player, there is a 5% chance you will steal an item from their inventory. |
| **BedBomb** | Beds explode as soon as you try to sleep in them. |
| **Birds** | All players can fly. |
| **Blocked Blocks** | You cannot break blocks that you have placed yourself. |
| **Bloody Diamonds** | Mining a diamond deals half a heart of damage to you. |
| **Bloody Enchantments** | Every enchantment deals half a heart of damage to you. |
| **Bowless** | Bows do not exist with this setting. |
| **Cold Weapons** | You cannot enchant your weapons with Fire Aspect or Flame. |
| **CooldownPvP** | There is a cooldown between hits—similar to Minecraft versions 1.9 and above. |
| **CutClean** | Ores and food are dropped smelted/cooked. |
| **Daylight Circle** | Every time a player dies, the time of day switches between day and night. |
| **DecreaseHearts** | Every player has a maximum health equal to the number of players remaining. |
| **Diamondless** | Diamond ore drops are disabled, but you receive one diamond per kill. |
| **Dolphins** | You move faster in water. |
| **DoNotDisturb** | No other player can attack you while you are in a fight. |
| **Double Jump** | You can use double jumps. |
| **Dragons** | The host can spawn an Ender Dragon at every player's location using `/dragons`. |
| **Elimination** | Every 10 minutes, the player with the fewest hearts dies. If multiple players have the same health, no one dies. |
| **Enchanted Death** | Enchantment tables are disabled. Kills improve the enchantments on your equipment. |
| **ExplosiveArrows** | Arrows explode when they hit their target. |
| **Fireless** | No fire or lava damage. |
| **Fire Focus** | You deal double damage to other players if you are on fire or poisoned. |
| **Flower Power** | Breaking a flower drops a random item. |
| **GetAway** | You receive Speed II for 30 seconds after killing a player. |
| **Golden Heal** | Every player starts with a golden hoe that can be used once for a full heal. |
| **Goldless** | There are no gold ores, but you receive 8 gold per kill. |
| **Gold Digger** | Every ore also contains a gold nugget. |
| **Hastey Boys** | Every tool has Efficiency III and Unbreaking III. |
| **Healing Kill** | You receive full health after every kill. |
| **Heart Upgrade** | After every kill, your maximum number of hearts increases by one. |
| **Horseless** | Horse spawning is disabled. |
| **Hungerless** | You do not lose hunger. |
| **Hydrophobic** | You take damage while in water. |
| **Ice Core** | Lava below Y=16 is replaced by ice. |
| **IncreaseHearts** | All players start with only one heart. Every time a player dies, everyone else gains an extra heart. |
| **Instant Head** | Player heads can be traded for 10 seconds of Speed II via right-click. |
| **Interval Damage** | Every 10 minutes, every player takes 2 hearts of damage. |
| **Killswitch** | The killer of a player takes over the inventory of the deceased. |
| **Limited Inventory** | Only the hotbar can be used. The rest of the inventory is locked. |
| **MiddleEnchants** | Enchantment tables are disabled except for four located in the center. |
| **MobileCraftingTable** | Open your mobile workbench with `/workbench`. |
| **Multi Ores** | Ores drop a multiplied amount when mined (default 2x). |
| **Natural Regeneration** | Natural regeneration is enabled. |
| **NoClean** | You are invulnerable for 30 seconds after every kill. |
| **NoClean Plus** | Combines the NoClean, DoNotDisturb, and SafeLoot scenarios. |
| **NoEnchant** | No enchantments are possible. |
| **NoFall** | No fall damage. |
| **OnlyGoldenApples** | Every apple that drops is a golden apple. |
| **OpSword** | You can craft a Sharpness X diamond sword. |
| **Ore Apples** | Golden apples can be crafted with all types of ore blocks. |
| **Ore Limitations** | You can only mine 64 iron ores, 32 gold ores, and 16 diamond ores. |
| **Regenerating Kill** | After a kill, the player receives Regeneration II for 10 seconds. |
| **Rodless** | Fishing rods cannot be crafted. |
| **Safeloot** | After a kill, only the killer (and their team) can pick up the items of the dead player or open their TimeBomb for 30 seconds. |
| **Shared Heal** | If a player is healed, their teammates are healed as well. |
| **Show Health** | Displays the current health of every player above their head. |
| **Sniper** | Arrows shot from more than 30 blocks away deal double damage. |
| **Soup** | Soup healing is enabled (2 hearts). |
| **Switcheroo** | When hitting a player (arrow, egg, etc.), the positions of both players are swapped. |
| **Timber** | Breaking a tree with an axe fells the entire tree immediately. |
| **TimeBomb** | After a kill, a chest spawns with your opponent's items. It explodes after 30 seconds. In Timolia-hosted UHCs, you receive a ready-crafted Golden Head instead of a player head. |
| **Toplist** | Every 10 minutes, you regenerate 2 health and receive 2 extra hearts until you take damage. Then you must kill a player to receive health again. |
| **Tracker** | You receive a compass at the start of the game that can lead you to the nearest player. |
| **xRay-Pickaxe** | This pickaxe can be crafted and turns all surrounding stone into glass for a short time via right-click. |
| **Zombie Death** | When a player dies, a zombie in diamond armor spawns at their location, which must be killed to loot the items. |

## Premium Features
- After being eliminated, Premium players have the option to continue spectating the game.
- **[MVP](/ranks/premium/)** or higher: With `/latescatter`, you can join the server and participate in the game until the end of the protection period (default 20 minutes).
- Be displayed in the standard color with `/togglerank`.

You can read about which Premium rank offers which advantages [here](/ranks/premium/).

## Commands
| Command | Function |
| :--- | :--- |
| `/backpack`, `/bp` | Enables a shared inventory in Team UHCs |
| `/config` | Lists the current configuration of the UHC |
| `/coordinates`, `/coords` | Sends a message to your teammates with your coordinates |
| `/fix`, `/stuck` | Allows you to reload all players |
| `/helpop <Message>` | Allows you to ask the team questions about the UHC at any time |
| `/invite <Player>` | Invites players to your team in Team UHCs |
| `/inviteaccept <Player>` | Accepts team invitations in Team UHCs |
| `/nightvision` | Keeps your vision clear for the entire UHC |
| `/playerinfo <Player>` | Shows a player's kills and potentially who they were killed by |
| `/rules` | Shows all the rules for the current UHC |
| `/scenarios`, `/scn`, `/sc` | Shows the active scenarios |
| `/showhealth <Player>`, `/h <Player>` | Shows the health of the specified player |
| `/topkills`, `/tk`, `/kt` | Shows the player(s) with the most kills |
| `/teaminfo <Player>`, `/ti <Player>` | Shows the members of a team |
| `/uhc` | Lists all the commands you can execute |

**[MVP](/ranks/premium/) only**:
| Command | Function |
| :--- | :--- |
| `/latescatter` | Allows you to join the UHC during the protection period even after the game has started |
