You can join our Discord server via [https://timolia.de/discord](https://timolia.de/discord). There you can communicate with your friends and the community. Additionally, you can get support, report bugs, participate in giveaways, and much more.
Below is a rough overview of our Discord; all further information can be found directly on the Discord in the channels under the "Information" category.

## Verification on Discord
- When you join the community Discord, you are initially unverified. By verifying with your Minecraft account, you receive a special role with your ingame rank and additional permissions on the Discord. You will also receive the **"Discord Crew"** subtitle ingame!
- Once you have joined the Discord server, you can send a private message to the "**Timolia Bot**". Use the command `/register <ingamename>` to receive a confirmation prompt in the ingame chat (provided that the account is currently online on the network).

## Discord Rules
The rules of the Discord server can be found in the [#rules](https://discord.com/channels/407554118887014402/407565978025852929) channel on Discord or read [here](/rules/discord/).

## Support
We offer support on this server in chat form. If community support cannot resolve a question, or if the request involves confidential information, a support ticket can be opened in `#support`. In this private chat, the team will attempt to resolve your concern and assist you further.

## Reporting Bugs
Bugs can be reported in the `bug-reports` channel.

The following information is important so that the bug can be fixed:
*   **Description:** What exactly is the bug/error?
*   **Time:** When did the bug occur? (e.g., "on April 20th at approx. 5:00 PM")
*   **Location:** Where did the bug occur? (e.g., "on server games01". The current server is displayed in the Tab list)
*   **Technical Details:** Which Minecraft version are you playing on? Are you using modifications? If so, which ones?
*   **Trigger:** Did the bug occur after a specific action?
*   **Error Messages:** Did you receive any error messages? If so, what did they say?

> If you have a screenshot or a video recording of the error, please attach it as well. This is often very helpful for us. Please upload these to online platforms, such as Imgur for photos or YouTube for videos.
